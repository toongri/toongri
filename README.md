# 🔭 Intro.
팀에게 가치있는 과제를 찾고, 이를 개선하기 위해 노력하는 개발자 김준형입니다.

3년차 Go 백엔드 개발자로, 중고 의류 이커머스 도메인에서 상품 관리 및 물류 처리 시스템을 설계하고 구현해왔습니다. LLM 기반 메타데이터 추론 시스템을 개발하여 검수 처리량 5배 향상 및 월 1,600만원 운영 비용을 절감한 경험이 있습니다.

비즈니스에 몰입해서 제품을 성장시키는 재미를 즐깁니다. 검수 조건 변경이 매번 배포를 기다려야하는 병목을 발견했을 때 공정 매니저님과 Rule Engine을 사이드 프로젝트로 먼저 검증한 뒤 실무에 도입하기도 했습니다.

최근에는 AI와 협업하며 업무 생산성을 높이는 방법에 대해 관심이 많으며 지인들과 케이스 스터디를 통해 함께 에이전트 도구 숙련도를 높이고 있습니다.
